What is Base64?

Base64 is a way to represent binary data using only 64 ASCII characters (A–Z, a–z, 0–9, + and /). It's used whenever binary data needs to travel through text-only systems — email attachments, data URIs, JSON payloads, basic-auth headers.

Important: Base64 is NOT encryption

Encoding is not the same as encrypting. Anyone with the Base64 string can decode it back to the original. Never use Base64 to "hide" passwords, API keys or sensitive data.

Common uses

  • Embedding images directly in CSS or HTML via data:image/png;base64,...
  • HTTP Basic Authentication headers
  • Encoding binary attachments in JSON or XML
