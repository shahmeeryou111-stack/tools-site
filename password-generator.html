Frequently Asked Questions

How long should my password be?

For modern security, use at least 16 characters. Add length before adding complexity — a 20-character random password is exponentially stronger than an 8-character one.

Is this password generator safe?

Yes. The generator runs 100% in your browser using the cryptographically secure Web Crypto API. Nothing is sent over the network.

Can someone guess a generated password?

A 16-character random password using all 4 character sets has more than 10^31 possible combinations — uncrackable by brute force.

Should I use the same password everywhere?

Never. Use a unique password for every account, and store them in a password manager.
