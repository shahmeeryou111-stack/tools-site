What to include in a bug report

  • The tool you were using (e.g. Password Generator)
  • Your browser and version (e.g. Chrome 124 on Windows 11)
  • The steps that reproduce the issue
  • What you expected to happen, and what actually happened

Feature requests

If you have an idea for a new tool, tell us:

  • What problem does it solve?
  • How do you currently solve it?
  • Roughly how often do you need it?
