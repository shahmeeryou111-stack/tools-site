Frequently Asked Questions

Does this follow shortened URLs?

No — for privacy and safety, the analyzer only inspects the URL string itself without making any network request.

Can it detect phishing?

It flags non-HTTPS and shows the real domain so you can spot impersonation, but you should still use security software.

What does the path show?

Everything after the domain, before the query string — the page or resource location on the server.

Is HTTPS always safe?

HTTPS only means the connection is encrypted, not that the site itself is trustworthy. Always check the domain too.
