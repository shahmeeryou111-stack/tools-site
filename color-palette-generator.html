Frequently Asked Questions

What's the difference between the modes?

Analogous = sibling hues for soft looks. Complementary = high-contrast pairs. Triadic = three balanced hues. Monochrome = one hue, many shades.

Can I use these palettes commercially?

Yes — colors aren't copyrightable. Use them freely for any project.

How do I find my brand's base color?

Start with the hue that represents your brand emotion (blue=trust, red=energy, green=growth). Then build harmony around it.

Are the hex codes accessible?

Always check contrast ratios (WCAG AA = 4.5:1 for text) when pairing colors for UI.
