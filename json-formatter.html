Frequently Asked Questions

    
What is JSON?
JSON (JavaScript Object Notation) is a lightweight data-interchange format, widely used by web APIs.
Is my JSON safe?
Yes — formatting runs in your browser. Sensitive payloads never leave your device.
Why won't my JSON validate?
Common issues: trailing commas, single quotes (JSON requires double), or missing brackets. The error message points to the location.
Can I format JSON5 or JSONC?
Currently strict JSON only. Remove comments and trailing commas first.
