Frequently Asked Questions

Will compression hurt image quality?

At quality 0.7–0.85, file size drops dramatically with no visible difference. Tune to taste.

Which format should I use?

WebP for the web (smallest size), JPEG for photos with broad compatibility, PNG when transparency matters.

Is there a file size limit?

No upload, no server limit. Browsers comfortably handle images up to 50MB.

Does this work on mobile?

Yes — fully responsive. Compress photos directly from your phone.
