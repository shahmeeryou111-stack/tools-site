Frequently Asked Questions

What is Base64?

Base64 is an encoding scheme that represents binary data as ASCII text using 64 characters. Common in email, JWTs, and data URIs.

Is Base64 encryption?

No — it's encoding, not encryption. Anyone can decode it. Never use Base64 to hide secrets.

Why does my decoded text look garbled?

The input may be invalid Base64. Check for whitespace, line breaks, or missing padding (=).

Does it work with images?

This tool is text-only. For image-to-Base64, use a dedicated data URI tool.
